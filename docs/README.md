🎄우테코 식당 12월 이벤트 플래너 프로그램🎄
===

<br>
2023년 12월, 우테코 식당에서 이벤트를 개최합니다.<br>
많은 혜택이 준비되어있으니, 받을 수 있는 혜택을 이벤트플래너 프로그램으로 확인해보세요!<br>
방문날짜와 주문할 메뉴를 고르면 받을 수 있는 혜택을 미리 알아볼 수 있습니다<br>

- - -
🎁 혜택 안내
---
### ⏲ 이벤트 기간
혜택은 크리스마스 디데이 할인을 제외하고 2023.12.1 ~ 2023.12.31 동안 적용됩니다. <br>
<br>

### ❕ 혜택 적용 조건
총주문 금액 **10,000원 이상**부터 아래의 이벤트가 적용됩니다<br>
<br>

### 1. 크리스마스 디데이 할인
1,000원으로 시작하여 크리스마스가 다가올수록 날마다 할인 금액이 100원씩 증가합니다!<br>
총주문 금액에서 해당 금액만큼 할인됩니다.<br>
(e.g. 시작일인 12월 1일에 1,000원, 2일에 1,100원, ..., 25일엔 3,400원 할인)<br>
>이벤트 기간: 2023.12.1 ~ 2023.12.25<br>

###  2. 평일 할인(일요일~목요일)
디저트 메뉴 1개당 **2,023**원 할인

### 3. 주말 할인(금요일, 토요일)
메인 메뉴 1개당 **2,023**원 할인

### 4. 특별 할인
이벤트 달력에 별이 있으면 총주문 금액에서 **1,000**원을 할인

### 5. 증정 이벤트
총주문 금액이 12만 원 이상일 때, **샴페인 1개**를 증정

<br>

### 🔰 혜택 금액에 따른 12월 이벤트 배지 부여 이벤트
**총혜택 금액**에 따라 다른 이벤트 배지를 부여합니다.<br>
이 배지는 2024 새해 이벤트에서 사용될 수 있습니다.<br>
배지에 따라 새해 이벤트 참여 시, 각각 다른 새해 선물을 증정할 예정입니다.<br>
- 5천 원 이상: 별 ⭐
- 1만 원 이상: 트리 🎄
- 2만 원 이상: 산타 🎅

```

```
방문할 날짜(1~31)를 숫자로 입력해주세요!<br>
<br>
입력에시)
```

```

### 2. 메뉴를 골라주세요

```
주문하실 메뉴를 메뉴와 개수를 알려 주세요. (e.g. 해산물파스타-2,레드와인-1,초코케이크-1)
```
메뉴판을 참고하여 정확한 이름의 메뉴와 개수를 입력해주세요!<br>

📢 주의<br>
- 개수는 1개 이상으로 지정하여 입력해주세요.
- 형식에 맞춰 입력해주세요.
- 메뉴는 중복하여 입력하지 마시고, **하나로 묶어서** 입력해주세요.
- 메뉴는 한 번에 **최대 20개**까지만 주문할 수 있습니다.
- 음료만 주문은 불가능합니다.

<br>
입력예시)

```
티본스테이크-1,바비큐립-1,초코케이크-2,제로콜라-1
```
```

```
